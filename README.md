## 🚀 核心能力

### 核心功能
- ✅ 多格式支持：[30+ 模型格式（GLTF/OBJ/FBX/GLB/RVT/IFC等）](http://editor-doc.astraljs.com/guide/f7smai4w/#%E5%9F%BA%E7%A1%80%E6%93%8D%E4%BD%9C%E5%8C%BA)
- ✅ BIM模型轻量化展示（RVT/IFC）
- ✅ CAD图纸解析（DWG/DXF）
- ✅ 场景分包存储与加载
- ✅ 动画编辑器

## ⚡ 快速开始

### 前置需求
- Node.js ≥ 18.x
- Yarn

### 本地运行
```bash
git clone https://github.com/mlt131220/Astral3DEditor.git
cd Astral3DEditor
yarn install
yarn run dev
```

### 生产构建
```bash
yarn run build
```
