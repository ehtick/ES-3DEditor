本指南将向您说明 `Astral 3D` 项目的文件结构。

`Astral 3D`代码是以`Monorepo`的方式管理的，它的文件结构是这样的：

::: file-tree

- .git/
- .idea/
- common \# 通用代码包
    - build/  \# vite构建相关
- packages \# 主项目包
    - docs/  \# Astral 3D 文档
    - **editor/**  \# Astral 3D 编辑器
    - **sdk/**  \# Astral 3D 引擎
- .gitignore
- .npmrc
- CONTRIBUTING.md
- LEGAL.md
- LICENSE
- package.json
- pnpm-lock.yaml
- pnpm-workspace.yaml
- README.en.md
- README.md
  :::
